### About

I'm an engineer and founder building **AI-powered products** from idea to production. My work spans **applied machine learning**, **full-stack development**, and **serverless cloud infrastructure** with a focus on shipping real products that solve real problems.

I move across the stack: training and deploying ML models, designing APIs, building production UIs, and standing up the cloud infrastructure that holds it all together.

### Currently Shipping

- **[ApexLot](https://apexlot.vercel.app)** — Multi-mode photo search for motorsport and car events. CLIP-based visual search, OCR, color and plate filters. Built on Next.js, Vercel, ONNX, and Replicate.
- **[askabhi.ai](https://askabhi.ai)** — My AI sales agent, live and taking real calls. ElevenLabs Voice + knowledge base + Calendly handoff.
- **[FaceFind](https://facefind-eight.vercel.app)** — Face-recognition photo search at event scale. Migrated from a GCP VM to Cloud Run + Vercel + Upstash for serverless wins.
- **Hooten Young Marketing Hub** — AI-powered competitor intelligence for a premium spirits brand. RAG + multimodal Claude Vision.
