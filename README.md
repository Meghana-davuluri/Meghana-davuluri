## About

I'm an **engineer and founder** building AI-powered products from idea to production. My work spans **applied machine learning, full-stack development, and serverless cloud infrastructure** — with a focus on shipping real products that solve real problems.

I love taking ambiguous ideas, breaking them into the smallest thing that works, and getting it in front of real users fast. Most of what I build lives in production and serves real users today.

## Currently Shipping

- **[ApexLot](https://apexlot.vercel.app)** — Multi-mode photo search for motorsport and car events. CLIP-based visual search, OCR, color/plate filters. Built with Next.js + Vercel + ONNX/Replicate.
- **[askabhi.ai](https://askabhi.ai)** — Meghan, my AI sales agent. ElevenLabs Voice + knowledge base + Calendly handoff. Live and taking real calls.
- **[FaceFind](https://facefind-eight.vercel.app)** — Face-recognition photo search at event scale. My own product, built from idea to live. Migrated from GCP VM → Cloud Run + Vercel + Upstash for serverless cost and latency wins.
- **Hooten Young Marketing Hub** — AI-powered competitor intelligence for a premium spirits brand. RAG over Social Insider PDFs + multimodal Claude Vision for content pattern analysis.
